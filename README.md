# Test Page Repository

This is a test repository for experimenting with GitHub Pages and MkDocs deployment.

## About

This site serves as a testing ground for:

- MkDocs configuration and themes
- GitHub Pages deployment workflows
- Documentation structure and navigation
- Markdown formatting and features

## Getting Started

To run this site locally:

```bash
pip install mkdocs
mkdocs serve
```

The site will be available at `http://127.0.0.1:8000/`

## Deployment

This site is automatically deployed to GitHub Pages. To manually deploy:

```bash
mkdocs gh-deploy
```

## Built With

- [MkDocs](https://www.mkdocs.org/) - Documentation site generator
- [GitHub Pages](https://pages.github.com/) - Hosting platform

## Purpose

This is a test environment for learning and experimentation. Content may change frequently as new features and configurations are tested.
